2012 State of the Coast Conference

October 4, 2012

The CSS recently attended the 2012 State of the Coast Conference in New Orleans. This bi-annual conference was attended by around 800 people including Louisiana and the Gulf’s leaders in restoration, protection, politics, and education. Following the 2012 masterplan the conference had a buzz about it that was quite exciting. You can tell that things are about to change for the Gulf!

The CSS had a booth designed and run by our students. It included a dramatic vision of Louisiana as part of an infrastructure network that links us to the rest of the world. Additionally we featured a number of our projects on boards and the video screen.

At the Conference Jeff Carney presented a paper titled, “Achieving Stable Communities in an Unstable Landscape: The Louisiana Resiliency Assistance Program”. The presentation discussed the project with OCD-DRU that the CSS has been working on over the past few months. See more about the project here: http://resiliency.lsu.edu/

About the studio

CSS brings together disciplines that often work independently —such as designers, scientists, engineers, and planners—to respond to critical issues of affecting coastal regions.

Through trans-disciplinary design, research, community engagement, and education, the CSS uses innovative approaches to design and foster resilient coastal communities and ecosystems.
